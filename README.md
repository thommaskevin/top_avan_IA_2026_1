# Advanced Topics in Artificial Intelligence A Course Repository

## Overview

This repository contains educational materials for the **Advanced Topics in Artificial Intelligence A** course offered at the **Federal University of Rio Grande do Norte (UFRN)**, within the **Department of Computing and Technology (DCT)** at the **Center for Higher Education of Seridó (CERES)**.

The repository provides structured access to lecture slides, video lessons, and complementary materials aimed at supporting undergraduate students in developing modern Artificial Intelligence (AI) systems, with emphasis on Large Language Models (LLMs), Retrieval-Augmented Generation (RAG), intelligent agents, MLOps, and Edge AI.

## Course Objectives

### General Objective

- To enable students to design, develop, and deploy modern AI systems, including LLM-based applications, RAG pipelines, intelligent agents, and Edge AI solutions.

### Specific Objectives

- To use version control tools such as GitHub and manage virtual environments.
- To perform data analysis and preprocessing.
- To apply Feature Engineering techniques.
- To understand and implement Machine Learning models.
- To apply Large Language Models (LLMs) and fine-tuning strategies.
- To develop Retrieval-Augmented Generation (RAG) systems.
- To design and implement intelligent agents.
- To apply MLOps practices for deployment and monitoring.
- To develop AI solutions for Edge and embedded systems.

## Course Content

The course covers the following topics:

1. Data Science and Statistics fundamentals  
2. Feature Engineering techniques  
3. Machine Learning concepts and models  
4. Large Language Models (LLMs)  
5. Fine-tuning strategies for LLMs  
6. Retrieval-Augmented Generation (RAG)  
7. Intelligent agents and autonomous systems  
8. MLOps (deployment, monitoring, and lifecycle management)  
9. Edge AI and embedded intelligence  



## Repository Structure

The course content is organized into lectures ("Aulas"), each including:

- 📄 **Slides** (externally hosted)  
- 🎥 **Video lectures**  
- 📚 **Complementary materials** (e.g., articles, notebooks, exercises)

Each lecture provides:

- Title  
- Description  
- Learning resources  
- Additional supporting materials (when available)  


## How to Use

1. Clone this repository:

```bash
   git clone https://github.com/thommaskevin/advanced_ai_2026_1.git
````

2. Open the `index.html` file in your web browser.

3. Use the search bar to filter lecture topics.

4. Access the materials:

   * Click **Slides** to view presentation content
   * Click **Video** to watch lecture recordings
   * Use the **Material Complementar** dropdown for additional resources
